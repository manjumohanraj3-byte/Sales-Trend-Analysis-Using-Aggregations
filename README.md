# Task 6 - Sales Trend Analysis Using Aggregations

## Objective
Analyze monthly revenue and order volume from sales data using SQL aggregate functions.

## Tools Used
- MySQL Workbench 8.0 CE

## Dataset
online_sales table containing:
- order_id
- order_date
- product_id
- amount

## Tasks Performed
- Monthly revenue analysis using SUM()
- Monthly order volume analysis using COUNT(DISTINCT)
- Grouping data by year and month
- Sorting results using ORDER BY
- Finding top 3 months by sales

## Outcome
Successfully analyzed monthly sales trends and generated revenue and order volume reports using SQL.
